# Task Contract: PR15.1 — CI matrix across Node, Kafka & PostgreSQL

## Outcome
A GitHub Actions matrix continuously verifies build/check plus live storage and
transport suites across supported Node, Kafka and PostgreSQL versions.

## Scope
- In: `.github/workflows/ci.yml`; service containers for PG + Kafka; a kind-based
  k8s smoke job; `docs/dev/CI.md`.
- Out: performance/soak (PR12.1), release automation (PR16.1).

## Affected module
Repo CI; exercises all packages + test-shop.

## Acceptance — frozen
- [ ] Matrix: Node {22, 24} × build+check; PG {14,15,16,17} live storage
      conformance; Kafka {3.7.x, 4.x} live transport conformance.
- [ ] A kind cluster job runs `k8s:deploy` + `k8s:test` smoke (subset acceptable
      if time-bounded) and passes.
- [ ] PRs are gated on the matrix; `main` is green.

## Required tests
The existing live suites (`test:live`) wired to CI service containers.

## Dependencies
PR5.1 (migration readiness helps live PG), PR6.1 (multi-partition). **Priority**
P0 · **Size** M · **Blocks stable** yes.

## Docs
`docs/dev/CI.md`.

## Stop conditions
If a supported version fails a real invariant → record as defect, do not drop the
version silently.
